Function slos::filesystem::fopen[−][src]

pub fn fopen<'a>(path: &str) -> Result<&'a mut dyn FsFileHandle, FsError>
Expand description

Open a file somewhere on the filesystem, returning a handle

The path is normalized and segmented before being used to call the FilesystemBase::node_at_path method to find the file in question. Errors from that are bubbled, if that function returns a non-file node then we return an Err(FsError::FileNotFound), otherwise we try to get a handle on that file by returning the Result from FsFile::open.
